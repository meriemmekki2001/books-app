## This was part of my master's degree at CY Tech - CY Cergy Paris Université
## Project Overview

This project is a minimalist web application that connects a **MongoDB Atlas NoSQL database** to a **backend API** and a **frontend interface**.  
The application allows users to manage and interact with a collection of books.

### Learning Objectives

The main goals of this project were:

- Connect a minimalist **Node.js backend** to a **MongoDB Atlas NoSQL database**.
- Implement the **MVC (Model–View–Controller)** architecture in a Node.js application.
- Migrate and use a **real dataset**, converting data from **CSV format to MongoDB Atlas**.

### Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Architecture:** MVC (Model–View–Controller)  
- **Data Source:** CSV dataset migrated to MongoDB Atlas
